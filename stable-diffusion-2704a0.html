EmebddingEmbedding

Currently, there are more and more negative embedding, while many are also very good and easy to use. However, almost all of them currently have a big problem... They change the main or initial artstyle of the used model. Best example would be my bad_prompt_version2 Negative Embedding. It helps enormously with the quality of an image, but drastically changes the artstyle of the model. That's not the point of it. For this reason, I have now trained my new Negative Embedding negative_hand!目前，负嵌入越来越多，而许多也非常好且易于使用。但是，目前几乎所有人都有一个大问题......它们更改所用模型的主要或初始艺术风格。最好的例子是我的bad_prompt_version2负嵌入。它极大地帮助了图像的质量，但极大地改变了模型的艺术风格。这不是重点。出于这个原因，我现在已经训练了我的新负嵌入negative_hand！

So what is negative_hand?

negative_hand is supposed to fix the said issue. That means it should improve the quality of the image, but without changing the initial artstyle of the model.

Pro:

  • The artstyle of the model can be used without any problems and without possible artstyle changes.
  • The quality of the image and incorrect anatomy like hands are improved.

Con:

  • Since this embedding cannot drastically change the artstyle and composition of the image, not one hundred percent of any faulty anatomy can be improved.

Usage

To use this embedding you have to download the file aswell as drop it into the "\stable-diffusion-webui\embeddings" folder.要使用此嵌入，您必须下载文件并将其放入“\stable-diffusion-webui\embeddings”文件夹中。

Please put the embedding in the negative prompt to get the right results!

For special negative tags such as "malformed sword", you still need to add them yourself. The negative embedding is trained on a basic skeleton for the negative prompt, which should provide a high-resolution image as a result.对于“malformed sword”等特殊负面标签，您仍然需要自己添加它们。负嵌入是在负提示的基本骨架上训练的，因此应该提供高分辨率图像。
